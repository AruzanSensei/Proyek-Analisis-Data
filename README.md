# Proyek Analisis Data - Dicoding
 
## Setup environment
```
conda create --name main-ds python=3.9
conda activate main-ds
pip install streamlit pandas scipy matplotlib seaborn jupyter babel numpy
```

## Run steamlit app
```
streamlit run dashboard.py
```
